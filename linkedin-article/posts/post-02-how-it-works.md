I built a 7-agent AI marketing analytics team.
I typed one question. 7 agents fired. One answer came back in under a minute.

Building this isn't about making agents smarter.
It's about making them more restricted.

– One Orchestrator receives every question
– Routes it to the right specialist
– Each agent owns one job — blocked from doing anything else

The Orchestrator doesn't touch data.
The Data Agent doesn't analyse.
The Budget Agent doesn't touch creative.

That's the Supervisor pattern.
Structured delegation. Every agent accountable for exactly one thing.

This isn't theory. It's a working system.

Stack: Python + Claude Sonnet API. SQLite. Streamlit. No cloud. Runs on a laptop.
7 agents. Under 1 minute.
